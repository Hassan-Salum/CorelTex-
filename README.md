## What CorelTex Is Building

CorelTex focuses on **AI memory, identity, and trust infrastructure**, including:

- **Sovereign Memory Protocol**  
  Long-term AI memory owned by the user, not the model provider.

- **HybridRAG Architecture**  
  Combining Knowledge Graphs (structure, logic, provenance) with Vector Retrieval (speed and semantic similarity).

- **Continuum Memory Systems**  
  Memory that evolves over time: retention, decay, consolidation, and mutation.

- **Identity & Provenance Layer**  
  Cryptographic signing of memories and agent actions using decentralized identifiers (DIDs).

## Why CorelTex Exists

Modern AI systems suffer from three structural failures:

1. **Amnesiac Intelligence**  
   Context disappears after each session.

2. **Black-Box Memory**  
   No way to verify where answers come from.

3. **Vendor Lock-In**  
   Memory owned by centralized platforms, not users.

CorelTex exists to solve these problems at the infrastructure level.

## Core Principles

- **Sovereignty**

- **Provenance**

- **Persistence**
